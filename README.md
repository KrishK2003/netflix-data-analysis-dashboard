# Netflix Data Analysis Dashboard

## Overview
This project presents an interactive Tableau dashboard that analyzes Netflix movies and TV shows across countries, genres, ratings, and release years. The dashboard helps uncover trends in content distribution, genre popularity, and platform growth through data visualization and storytelling.

## Tools Used
- Tableau
- Data Visualization
- Data Analysis

## Key Features
- Interactive dashboard with dynamic filters
- Analysis of Netflix content by country and genre
- Content rating distribution visualization
- Release year trend analysis
- User-friendly visual storytelling

## Project Highlights
- Developed an interactive Tableau dashboard to analyze Netflix movies and TV shows across countries, genres, ratings, and release years.
- Created dynamic visualizations to explore content distribution, rating categories, and genre popularity trends.
- Built interactive filters and dashboards to enable deeper exploration of Netflix content data.
- Identified key insights related to content growth, geographic distribution, and viewer rating patterns.
- Utilized data storytelling techniques to present findings through intuitive and user-friendly visualizations.
